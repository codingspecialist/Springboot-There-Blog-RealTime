# 스프링부트 블로그 프로젝트 - 수업중 레포 (실시간 코드 반영)

## 1. 기술스택
- JDK 11
- Springboot 2.X 버전
- JPA
- H2 인메모리 디비 - 방언 MySQL
- JSP
- Security
- AJAX
- JSoup
- 부트스트랩
